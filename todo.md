# 待办记录

用于临时记录后续要集中处理的小问题和改进点。

## 待处理

- [ ] 精简置顶模式：支持从当前大屏使用模式切换到小窗模式，窗口可缩小并保持桌面置顶，类似视频播放器悬浮窗。小窗中只保留最精简功能，例如当前日期/当前记录概览、快速新增记录入口、开始/结束时间与工作内容等核心输入；不展示完整统计、管理和导出等复杂页面。需要评估窗口尺寸、置顶开关、返回完整模式、数据保存和 macOS/Windows 窗口行为差异。
- [ ] 开发维护：调整验证策略，纯 Markdown 文档修改（例如 `README.md`、`todo.md`、`log.md`、`AGENTS.md`）不再强制执行 `npm test` 和 `npm run build:mac`；只有代码、样式、Electron 配置、打包配置或影响运行行为的改动才需要测试和重新打包。

## 个人时间管理工具方向

- [ ] 当前任务计时模式：支持选择工作类型和关联工作后开始计时，结束时自动生成记录，并允许补写工作内容和备注。
- [ ] 时间冲突和空白提醒：检测当日记录中的时间重叠、结束时间异常和未记录空档，帮助发现记录遗漏。
- [ ] 今日目标 / 计划：支持维护当天 3-5 个计划事项，完成后可一键转为工作记录，未完成事项可顺延或关闭。
- [ ] 周/月复盘：基于现有统计生成周报、月报和复盘草稿，突出总时长、类型占比变化、重点工作和备注记录。
- [ ] 标签系统：为记录增加灵活标签，例如深度工作、会议、返工、被打断、等待他人等，用于分析时间质量。
- [ ] 专注度和打断记录：支持记录主观专注度、打断次数、是否计划内工作，帮助区分时间长度和时间质量。
- [ ] 常用模板：支持常用工作内容、备注模板、最近使用工作项和一键复用上一条记录，降低录入成本。
- [ ] 数据备份和导入导出：支持手动备份 `daily-data.json`、导出 CSV、导出完整 Markdown、导入旧数据和数据校验修复。
- [ ] 个人时间仪表盘：在完整模式首页展示今日已记录时长、当前工作、本周趋势、最近 7 天平均时长和深度工作占比等关键指标。
