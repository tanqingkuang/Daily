## V1.0.5

- 日期：2026-06-16
- 变更内容：统计分析新增类型和关联工作复选筛选；取消勾选后对应行保留显示但置灰并添加删除线；统计总时长、类型占比、项目耗时和 Markdown 导出均按已勾选记录重新计算。
- 变更内容：新增「当日时间线」合计时长显示，按小时和分钟展示，例如 `3h20min`。
- 变更内容：新增 GitHub Actions 手动构建流程，可生成 macOS app zip 与 Windows exe 构建产物。
- 影响性：只影响统计展示、导出内容和构建获取方式；不影响 `daily-data.json` 格式。

## V1.0.4

- 日期：2026-06-16
- 变更内容：时间输入支持 `9:00`、`9：00`（全角）、`9:0`、裸数字等写法，提交时统一归一化为 `HH:MM`。
- 变更内容：新增记录成功后把结束时间同步为下一条的开始时间，便于连续录入。
- 变更内容：右侧「今日概览」下方新增只读「当日时间线」，压缩展示当日记录，不含编辑/删除操作。
- 变更内容：新增记录后停留在记录页，不再自动跳转到时间线。
- 变更内容：导航与视图标题「今日时间线」改为「时间线」，支持选择任意日期。
- 影响性：只影响记录录入和页面展示；不影响 `daily-data.json` 格式。
